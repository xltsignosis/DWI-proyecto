# MaquilaControl

Plataforma web SaaS para el control de producción por lotes y nómina automática en maquiladoras de confección en México.

## Sobre el proyecto

MaquilaControl nació de entrevistas directas con operadores, supervisores y administradores de maquiladoras en la región de Tehuacán, Puebla. El sistema resuelve el problema central de discrepancias en el conteo de piezas por lote y el cálculo manual de nómina por destajo.

Desarrollado como proyecto académico para la materia **Desarrollo Web Integral** en la Universidad Tecnológica de Tehuacán (UTTehuacán).

## Stack tecnológico

| Capa | Tecnología |
|---|---|
| Frontend | Next.js 14 + Tailwind CSS |
| Backend | Node.js + Express |
| Base de datos | PostgreSQL (Supabase) |
| Autenticación | JWT + bcrypt |
| Deploy frontend | Vercel |
| Deploy backend | Render |
